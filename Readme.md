#캡스톤 프로젝트 소개 웹 (ECS + Docker, 로컬)
본 저장소는 캡스톤 프로젝트 단일 소개 페이지와 API를 제공합니다.

## 1. 작품 주제(subject)
-제목: KDUKIT
-요약: 하나의 어플리케이션에서 필요한 모든 정보를 제공 및 익명 커뮤니티의 익명성으로 인한 단점 제한.

## 2. 실용적 근거(rationale)
-문제:익명 커뮤니티의 익명성으로 인한 불필요한 분쟁 및 정보오염 발생 가능, 여러 웹 사이트를 방문해야하는 번거로움 존재
-근거:에타의 경우 시간표 설정에 번거로움 존재종합된 정보를 가진 앱의부재
-가치: 시간 절약, 접근성 향상

## 3. 핵심 기능(features)
-기능 1: 시간표 관리
-기능 2: 중고장터
-기능 3: 동아리 모집
-기능 4: 분실물 찾기
-기능 5: 셔틀버스 잔여좌석 확인
-기능 6: 택시 동승자 구인
 
 ## 4. 구현환경(enviroment)
 -프론트엔드: javascript,typescript
 -백엔드: React,firebase
 -도구: git, VS code, Expo
 -배포: AWS ECS

 ## 5. 팀원 구성 및 역할
 - 김영한: 조장 / ppt 및 보고서 제작, 프론트엔드 개발
 - 김동주: 제작 총괄 / 백엔드 개발
 - 김도환: 백엔드 개발 / 제작 보조

 ## 6. 실행 방법
docker compose up --bulid -d
